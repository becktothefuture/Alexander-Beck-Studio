**Goal:** Connect to channel "ffzut1cu" and rebuild current design with meticulous auto-layout at 1120px width

## Step-by-Step Commands

### Step 1: Connect to New Channel
```
Join channel "ffzut1cu" in Figma
```

### Step 2: Get Current Design Information
```
Get document info from Figma
```

```
Get current selection in Figma
```

```
Read my design
```

### Step 3: Analyze Layout Structure
```
Get all frames and their properties from the current page
```

```
Get layout information for all selected frames
```

### Step 4: Create New Container Frame (1120px width)
```
Create a frame at (0, 0) with size 1120x800 named "Rebuilt Layout"
```

### Step 5: Set Auto-Layout
```
Set layout mode to VERTICAL for frame "Rebuilt Layout"
```

```
Set padding for frame "Rebuilt Layout" to top: 0, right: 0, bottom: 0, left: 0
```
