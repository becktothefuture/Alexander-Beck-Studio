# Figma MCP Test Commands

**Test: Create a red auto-layout frame with white Comic Sans text**

## Step-by-Step Commands (Run in Cursor Chat)

### 1. Join the Channel
```
Join channel "qhtxhv3o" in Figma
```

### 2. Create Auto-Layout Frame
```
Create a frame at position (100, 100) with size 400x200 named "Test Frame"
```

### 3. Set Auto-Layout Mode
```
Set layout mode to VERTICAL for the frame I just created
```

### 4. Set Fill Color to Red
```
Set fill color of the frame to rgba(255, 0, 0, 1)
```

### 5. Create Text Inside Frame
```
Create text "it fucking works" at position (120, 150) inside the frame
```

### 6. Set Text Color to White
```
Set fill color of the text node to rgba(255, 255, 255, 1)
```

### 7. Set Font to Comic Sans
```
Set font family of the text to "Comic Sans MS"
```
