# Config files

- `default-config.json` — runtime settings for the homepage simulation (copied to `public/config/default-config.json` + `public/js/config.json`).
- `text.json` — runtime copy for UI strings (copied to `public/config/text.json` + `public/js/text.json`).
- `portfolio-config.json` — carousel tuning (copied to `public/config/portfolio-config.json` + `public/js/portfolio-config.json`).
- `portfolio-data.json` — portfolio projects (copied to `public/config/portfolio-data.json` + `public/js/portfolio-data.json`).
